Site de streaming

Pour créer le site web demandé, nous avons d'abord déterminé les attentes du client pour son site de streaming vidéo.

Nous avons ensuite rédigé un rapport détaillant les fonctionnalités techniques requises.

Charte graphique : Nous avons élaboré un document rassemblant toutes les informations sur le logo, les polices de caractères, les tailles et les couleurs.

Le maquettage a permis de présenter une première version de notre site avec les différentes couleurs et polices.

Lors de cette étape, nous avons pu faire des ajustements concernant les positionnements, les alignements et la responsivité à prévoir.

Ces ajustements ont été réalisés grâce à une communication directe avec le client, assurant ainsi un site répondant à ses attentes.

Cette étape finale a consisté à coder le site en HTML, CSS, et JavaScript en s'appuyant sur le recueil des besoins, la charte graphique et la maquette.

Nous avons ainsi pu livrer un site fonctionnel et adapté aux besoins du client.
