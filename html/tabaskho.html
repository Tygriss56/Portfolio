Site de Streaming

Description

Dans le cadre de ce projet de première année, nous avons d'abord analysé les besoins d'un client fictif pour son site de streaming vidéo afin de définir les fonctionnalités essentielles.
Nous avons ensuite rédigé un rapport technique détaillant les aspects à prendre en compte pour la réalisation du site.

En parallèle, nous avons élaboré une charte graphique, incluant le logo, les polices, les tailles et les couleurs, afin d'assurer une cohérence visuelle.

Enfin, un maquettage a été réalisé pour présenter une première version du site, ce qui nous a permis d'ajuster les éléments graphiques avant le développement.


Outils utilisés

Le développement du site a été réalisé en HTML et CSS, en prenant en compte le recueil des besoins, la charte graphique et la maquette. Nous avons utilisé GitHub pour la gestion de version et la collaboration. Cela nous a permis de livrer un site fonctionnel, parfaitement adapté aux attentes du client.

Acquis

Ce projet m'a permis de débuter mon apprentissage du HTML et du CSS, en appliquant ces technologies pour créer un site fonctionnel. Il m'a également offert l'opportunité de travailler en équipe, ce qui a renforcé mes compétences en collaboration et en gestion de projet.
