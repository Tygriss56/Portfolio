Site de la PACT

Description

Ce projet universitaire consiste en la conception et le développement d’une application de gestion des zones et des représentations d’une salle de spectacle.
Cette application permet aux administrateurs de définir différentes zones dans la salle, d’organiser les représentations et de gérer les réservations.

Outils utilisés

Nous utilisons GitHub pour la gestion du code et la collaboration, garantissant un suivi efficace des versions et une intégration fluide du travail de chaque membre de l’équipe.
Le développement est réalisé en Java avec JavaFX, permettant de créer une interface utilisateur ergonomique et dynamique.

Acquis

Ce projet m’a permis de renforcer mes compétences en développement orienté objet grâce à Java et d’approfondir mes connaissances en modélisation d’interfaces graphiques avec JavaFX. J’ai également appris à travailler en équipe sur un projet informatique, en appliquant les bonnes pratiques de gestion de projet et de collaboration via GitHub.
